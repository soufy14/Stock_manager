# Mon Projet Stock Manager


**Stock Manager** est une application web simple conçue pour **gérer les stocks de produits**.  
Elle permet d’afficher, organiser et suivre les produits grâce à une interface claire et responsive, construite uniquement avec **HTML** et **CSS**.

---

##  Fonctionnalités

-  Affichage des produits sous forme de tableau  
-  Navigation intuitive entre les sections  
-  Design responsive et moderne  
-  Structure HTML claire et sémantique  
-  Mise en forme avec CSS (grilles, flexbox, couleurs, etc.)
